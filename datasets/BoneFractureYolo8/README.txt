## English Version

### Bone Fracture Detection

The goal of this project is to develop a computer vision model to detect bone fractures in upper limb radiographs. Initially, a binary classification model was trained to determine if a fracture was present or not. Later, the model evolved to YOLO, allowing for not only the identification of fractures but also the localization of the fracture on the image.

### Project Evolution

#### Phase 1: Binary Model
- Trained a binary CNN to classify images as "fracture" or "no fracture."
- Used a simple dataset for initial testing.

#### Phase 2: YOLO Model Implementation
- Transitioned to the YOLO model for object detection, including fracture localization on the images.
- Integrated a dataset from Kaggle with annotations for various fracture types.

For more details, visit [Português](README.pt.md).

## Versão em Português

### Detecção de Fraturas Ósseas

O objetivo deste projeto é desenvolver um modelo de visão computacional para detectar fraturas ósseas em radiografias dos membros superiores. Inicialmente, foi treinado um modelo binário para determinar se havia ou não uma fratura. Posteriormente, o modelo evoluiu para o YOLO, permitindo não só a identificação da fratura, mas também a localização da mesma na imagem.

### Evolução do Projeto

#### Fase 1: Modelo Binário
- Treinamento de uma CNN binária para classificar as imagens como "com fratura" ou "sem fratura."
- Utilização de um dataset simples para os primeiros testes.

#### Fase 2: Implementação do Modelo YOLO
- Transição para o modelo YOLO para detecção de objetos, incluindo a localização da fratura nas imagens.
- Integração com um dataset do Kaggle com anotações para diferentes tipos de fraturas.

Para mais detalhes, visite [English](README.en.md).
